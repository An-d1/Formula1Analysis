# Formula 1 Race Analysis Dashboard

This dashboard offers a variety of charts that are dynamically updated based on the users choice.
- First, choose a year, and based on the year the list of all disponible races for that year will apear.
- Secondly you have to chose one of the races that appeared on the dropdown.
- Finally, click 'Start Analysis' and the charts should appear along with personalized interpretation

This design makes the project scalable and dynamic on users needs. 
(The full analysis is only available for races from 2018 and onwards, as the data for older years is not available.)
